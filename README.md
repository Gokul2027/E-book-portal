# 📚 E-Book Portal

A web-based platform where users can browse, read, and manage digital books online.

## 🚀 Features

- 📖 Browse and read e-books
- 🔍 Search books by title/category
- 📂 Organized digital library
- 💾 Easy access to stored content

## 🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: (Flask / Node if used)
- Database: SQL / File-based storage

## 📂 Project Structure


├── index.html
├── books.html
├── reader.html
├── styles.css
├── scripts.js


## ⚙️ Setup Instructions

1. Clone the repository

git clone https://github.com/Gokul2027/E-book-portal.git


2. Open index.html in browser
